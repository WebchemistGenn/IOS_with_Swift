# Basic Swift

Swift를 공부하기위해 XCode로 간단한 앱을 만듭니다.

다음을 구현합니다.
1. Text, View를 이용한 간단한 페이지 구성
2. Input, Button을 이용한 Form 구성
3. Select, Checkbox, Radio 등의 더 복잡한 Form 구성
4. Navigation
5. Tab Navigation
6. Drawer Navigation
7. Swiper (배너)
8. Components (상세한 내용은 추후에 추가합니다.)


위의 내용을 모두 포함한 앱을 고민하여 기획, 개발합니다.
